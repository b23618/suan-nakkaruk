## 📁 Project Structure

### Application Files
```
app/
├── components/                 # Reusable React components
│   ├── Header.tsx             # Navigation header with mobile menu
│   ├── Hero.tsx               # Full-width image slider
│   ├── WhyChooseUs.tsx        # 4-column feature showcase
│   ├── FeaturedProducts.tsx   # Product grid with hover effects
│   ├── FarmStory.tsx          # About section with company values
│   ├── CustomerReviews.tsx    # Testimonials carousel
│   ├── LatestArticles.tsx     # Blog articles section
│   ├── Contact.tsx            # Contact methods and newsletter
│   └── Footer.tsx             # Footer with links and social media
├── page.tsx                   # Main home page (combines all sections)
├── layout.tsx                 # Root layout with metadata
└── globals.css                # Global styles and CSS variables
```

### Configuration Files
```
├── tailwind.config.ts         # TailwindCSS theme configuration
├── next.config.ts             # Next.js performance optimizations
├── tsconfig.json              # TypeScript configuration
├── postcss.config.mjs          # PostCSS configuration
└── package.json               # Dependencies and scripts
```

## 📊 Website Sections

1. **Header** - Fixed navigation with mobile menu
2. **Hero** - Full-width image slider with CTA buttons
3. **Why Choose Us** - 4-column feature showcase
4. **Featured Products** - Product grid with 4 items
5. **Farm Story** - About section with company values
6. **Customer Reviews** - Testimonials carousel
7. **Latest Articles** - Blog section with 3 articles
8. **Contact** - Contact methods and newsletter signup
9. **Footer** - Links, social media, and company info

## 🎨 Design System

**Colors:**
- Primary Dark: `#2D5A27`
- Primary Medium: `#6B8E23`
- Accent Gold: `#D4AF37`
- Cream: `#FFF8E8`

**Typography:**
- Font: Geist (Google Fonts)
- Headings: Bold, 600 weight
- Body: Regular weight

**Responsive:**
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## 🛠️ Tech Stack

- **Framework:** Next.js 16.2.9
- **Language:** TypeScript 5
- **Styling:** TailwindCSS 4
- **Animations:** Framer Motion 12.40.0
- **Icons:** Lucide React 1.17.0
- **Fonts:** Geist (Google Fonts)
